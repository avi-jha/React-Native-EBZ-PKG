# easebuzz-react-native-package

Easebuzz SDK for react native application.

## Installation

```sh
npm install easebuzz-react-native-package
```

## Usage


```js
import { multiply } from 'easebuzz-react-native-package';

// ...

const result = await multiply(3, 7);
```
